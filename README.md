## How to use

Clone this repo

```bash
git clone https://github.com/tim-soft/next-portfolio.git next-portfolio
cd next-portfolio
```

Install dependencies

```bash
yarn install
```

Run locally with hot module reloading

```bash
yarn dev
```

Deploy to Now

```
yarn global add now
now
```
